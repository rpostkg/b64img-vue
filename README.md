# b64img-vue

A basic Vue 3 page that encodes/decodes images to/from Base64. That's all.
The rest of the README is from the `vuejs/create-vue` template.

## Customize configuration

See [Vite Configuration Reference](https://vite.dev/config/).

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
